# Gradebook Management System (Python)

## Overview
The Gradebook Management System is a Python-based console application designed to manage student records efficiently.  
It demonstrates Object-Oriented Programming (OOP) principles, data persistence with JSON, and error handling.  
This project is ideal for showcasing practical coding skills.

---

## Features
- Add Student: Create new student records with subjects and marks.
- Remove Student: Delete student records by roll number.
- View Student Details: Display all subjects, marks, and GPA for a student.
- List All Students: Show all students currently in the gradebook.
- Save/Load Records: Persist data using JSON files.
- Update Marks: Modify marks for a specific subject.
- Top Performer: Identify the student with the highest GPA.
- Top K Performers: Display the top K students ranked by GPA.

---

## Technologies Used
- Python 3.x
- JSON for data storage
- OOP concepts (classes, methods, encapsulation)
- Error handling with try/except
